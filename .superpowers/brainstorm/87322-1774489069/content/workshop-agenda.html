Workshop Agenda — 100x Dev with Claude Code

1.5 hours total · Build Together Live format · Students finish with a working app

Segment 1 — 15 min

Intro: The 100x Dev Mental Model

You talk through how you actually think about working with Claude Code. Cover: superpowers plugin, what it changes, your workflow philosophy. Show the finished app so students know what they're building toward.

Segment 2 — 10 min

Setup Check + Repo Tour

Everyone clones the template repo, copies .env.example → .env, drops in their keys. You walk through SPEC.md and CLAUDE.md — explain how these two files are what drives the entire build.

Segment 3 — 20 min

Milestone 1: Rails Foundation

You build on screen, students follow. Tell Claude to implement milestone 1. Walk through what it's doing and why. Run docker compose up — app is live. First "whoa" moment.

Segment 4 — 20 min

Milestone 2 + 3: Schema + Plaid

Keep building together. Database schema goes in, then Plaid integration. Students connect their sandbox account and see real (fake) transactions appear. Second big moment.

Segment 5 — 15 min

Merge the Rest + See the Full App

Everyone pulls the complete branch to get milestones 4–8. Full analytics, recommendations, and UI are now live. Students explore their own app with their own data.

Segment 6 — 10 min

Q&A + What's Next

Open floor. Talk about extending the app, how to apply this workflow to their own projects, and where to learn more about Claude Code.

Checkpoint branches (in template repo)

checkpoint/milestone-1 · checkpoint/milestone-2 · checkpoint/milestone-3 · complete

If a student falls behind at any point, one git merge checkpoint/milestone-N catches them up instantly.
